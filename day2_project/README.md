## [6] 통계 검정 — t-test (유의수준 α = 0.05)
- 귀무가설 H0: 공항 출발과 일반 승차의 평균 팁 비율은 같다
- 공항 출발 평균 0.2030 (24,909건) / 일반 승차 평균 0.2496 (2,387,998건)
- t 통계량 = -30.2991,  p-value = 1.105e-200
- 해석: p < 0.05 이므로 귀무가설을 기각한다. 공항 출발 여부에 따라 평균 팁 비율에 통계적으로 유의한 차이가 있다.
